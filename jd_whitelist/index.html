About

This is a vehicle/object whitelist script,
which is used prevent some users from using vehciles/weapons while others have access.

you can whitelist any vehcile/weapon/weapon component/ped.
And if you have any issues adding setting up the permission you can join our discord and our support team will be happy to help.

Requirements

• FXServer (Setting up a server)

Main Features

• Vehicle Whitelisting
• Weapon Whitelisting
• Weapon Components Whitelisting.
• Ped Whitelisting.

Download & Installation

1. Download the files (Github)
2. Put the JD_WhitelistV2 folder in the server resource directory
3. Add this to your server.cfg
ensure JD_WhitelistV2
4. Open the config.lua in the JD_WhitelistV2 folder and update the settings.
5. Restart your server

Config.lua

Vehicles:

Config.Vehicles = {
  {GetHashKey("apc"), "jd.stafff","0"},	
  {GetHashKey("apc"), "jd.staff","1"},
}
To add more vehicles you can add just add on to the list.

GetHashKey("apc"): This is the has key of the vehcile. we're getting this with the GetHashKey but you could also add a vehicle hash if you prefer that.
jd.staff: This is the ace permission the player needs to be the driver of the vehicle.
1: The one will define that the vehcile will be deleted id a player doesn't have the right ace permission. If you don't want to delete the vehcile set this to 0

If you want to add more Vehicles you can find all of them here: Rage Wiki

Weapons:

Config.Weapons = {
  {GetHashKey("WEAPON_PISTOL"), "jd.staff"},
  {GetHashKey("WEAPON_PISTOL_MK2"), "jd.none"},
}
To add more weapons you can add just add on to the list.

GetHashKey("WEAPON_PISTOL"): This is the has key of the weapon. we're getting this with the GetHashKey but you could also add a weapon hash if you prefer that.
jd.staff: This is the ace permission the player needs to hold the weapon in their hands.
The weapons will get removed if the player doesn't have the ace permission

If you want to add more Weapons you can find all of them here: Rage Wiki

WeaponComponents:

Config.WeaponComponents = {
  {GetHashKey("COMPONENT_AT_PI_SUPP_02"), "jd.staff"},	
  {GetHashKey("COMPONENT_AT_PI_FLSH"), "jd.none"},
}
To add more weapons you can add just add on to the list.

GetHashKey("COMPONENT_AT_PI_SUPP_02"): This is the has key of the Weapon Component. we're getting this with the GetHashKey but you could also add a Weapon Component hash if you prefer that.
jd.staff: This is the ace permission the player needs to hold the weapon with the component in their hands.
The Weapon Component will get removed if the player doesn't have the ace permission

If you want to add more Weapon Components you can find all of them here: Rage Wiki

Peds:

Config.Peds = {
  {GetHashKey("mp_m_freemode_01"), "jd.staff"},	
  {GetHashKey("player_one"), "jd.none"},
}
To add more weapons you can add just add on to the list.

GetHashKey("mp_m_freemode_01"): This is the has key of the Ped. we're getting this with the GetHashKey but you could also add a Ped hash if you prefer that.
jd.staff: This is the ace permission the player needs to be this ped.
The ped will be reset to set to Franklin if the player doesn't have the ace permission

If you want to add more ped models you can find all of them here: FiveM Docs
